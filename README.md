## Overview

This React / Next Js project uses Open AI GPT-4  to summarize website text or article and was inspired by [Adrian Hajdin](https://github.com/adrianhajdin).

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## .env variables

```
NEXT_PUBLIC_RAPIDAPI_KEY=
NEXT_PUBLIC_BASE_URL=
NEXT_PUBLIC_API_HOST=
```
